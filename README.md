# STM32 Balance Car

这是一个基于 STM32 的两轮自平衡小车程序，主要用于学习嵌入式控制、姿态解算、电机 PWM 驱动、编码器测速以及平衡车 PID 控制逻辑。

当前工程偏向课程设计 / 实验项目 / 个人学习项目，代码中包含平衡控制、速度控制、转向控制、电机驱动、编码器读取、MPU6050 姿态读取、按键模式切换、串口通信等模块。

## 项目功能

- STM32 主控程序
- MPU6050 / DMP 姿态读取
- 陀螺仪角速度读取
- 编码器测速
- 电机方向控制与 PWM 输出
- 直立环 PD 控制
- 速度环 PI 控制
- 转向环 PD 控制
- 倾倒保护与 PWM 限幅
- 按键切换控制模式
- 串口通信接口，预留遥控 / OpenMV / 循迹等功能逻辑

## 控制思路

平衡车的核心控制流程大致如下：

1. 通过 MPU6050 获取车身姿态角，例如 Pitch、Roll、Yaw。
2. 通过陀螺仪读取角速度，用于辅助直立控制。
3. 通过左右轮编码器读取电机转速。
4. 在外部中断中周期性执行控制计算。
5. 使用直立环计算车身站立所需的基础 PWM。
6. 使用速度环修正前进、后退或静止时的速度误差。
7. 使用转向环根据 Z 轴角速度或转向指令生成左右轮差速。
8. 合成左右电机 PWM。
9. 对 PWM 进行限幅，避免输出过大。
10. 如果车身倾角过大，触发保护，关闭电机输出。

简化后的控制结构：

```text
MPU6050 姿态角 + 陀螺仪角速度
              ↓
          直立环 PD
              ↓
编码器速度 → 速度环 PI → PWM 基础输出
              ↓
转向输入 / Z 轴角速度 → 转向环 PD
              ↓
        左右电机 PWM 合成
              ↓
        限幅 / 倾倒保护
              ↓
          电机驱动输出
```

## 主要模块说明

### 1. 姿态检测

使用 MPU6050 获取车身姿态角和角速度。平衡车主要依赖 Pitch 角判断车身前后倾斜程度，并结合陀螺仪角速度进行动态修正。

### 2. 编码器测速

左右电机编码器用于获取车轮速度。速度环会根据编码器反馈判断当前运动状态，从而修正电机输出，减少小车前冲、后退或漂移。

### 3. 电机控制

电机控制部分主要包括：

- 电机方向 IO 控制
- PWM 占空比输出
- 左右电机差速控制
- PWM 输出限幅
- 倾倒保护关断

当前代码中电机 PWM 会进行限幅处理，避免控制量过大导致电机剧烈抖动或车辆失控。

### 4. 平衡控制

控制核心由三个部分组成：

#### 直立环

直立环通常使用 PD 控制：

```text
Vertical_PWM = Kp * (当前角度 - 机械中值) + Kd * 角速度
```

其中：

- `Kp` 决定车身偏离平衡点后的纠正力度；
- `Kd` 抑制车身快速摆动，提高稳定性；
- `机械中值` 是小车自然平衡时的角度偏移量。

#### 速度环

速度环通常使用 PI 控制：

```text
Velocity_PWM = Kp * 速度误差 + Ki * 速度误差积分
```

速度环用于控制小车前进、后退和静止状态下的速度趋势，防止小车一直向某个方向跑偏。

#### 转向环

转向环用于生成左右轮差速，让小车实现左转、右转或纠偏。

```text
左电机 PWM = 基础 PWM - 转向 PWM
右电机 PWM = 基础 PWM + 转向 PWM
```

## 调参建议

平衡车调参建议按下面顺序进行，不要一开始同时调所有参数。

### 1. 先调机械中值

机械中值决定小车认为“站直”的角度。如果机械中值不准，小车会一直向前或向后冲。

调试方法：

- 抬起小车，让轮子悬空；
- 观察 Pitch 角；
- 让小车接近自然直立状态；
- 记录此时角度，作为机械中值参考。

### 2. 再调直立环 Kp

现象参考：

- Kp 太小：车身软，扶不起来；
- Kp 太大：车身剧烈抖动，电机震动明显；
- Kp 合适：小车有明显回正趋势，但不过分震荡。

### 3. 再调直立环 Kd

现象参考：

- Kd 太小：来回摆动明显；
- Kd 太大：电机噪声明显，响应变僵硬；
- Kd 合适：能够抑制快速摆动，让车身更稳。

### 4. 最后调速度环

速度环主要影响小车是否会慢慢向前跑或向后跑。

现象参考：

- 速度环太弱：小车能站住但会慢慢漂移；
- 速度环太强：小车会前后抽动；
- Ki 太大：容易积分过量，导致突然冲出去。

## 安全注意事项

调试平衡车时建议：

- 第一次上电不要让轮子直接落地；
- 先悬空测试电机方向和编码器方向；
- 确认倾倒保护有效后再落地；
- 调参时逐步增加参数，不要一次改太大；
- 电池电量不足时，控制效果可能明显变差；
- 如果电机方向、编码器方向或 MPU6050 安装方向不一致，车会直接失控。

## 常见问题

### 小车一上电就疯狂转轮

可能原因：

- 电机方向反了；
- 编码器方向反了；
- MPU6050 安装方向或姿态角方向反了；
- 机械中值不正确；
- Kp 参数过大。

### 小车能站起来但一直往前跑

可能原因：

- 机械中值需要重新校准；
- 速度环参数不合适；
- 左右电机输出不一致；
- 编码器反馈方向有问题。

### 小车来回抖动很严重

可能原因：

- 直立环 Kp 太大；
- 直立环 Kd 不合适；
- 车架重心过高或结构松动；
- MPU6050 数据噪声较大。
